# AI视频制作_模仿参考库标签字典_V0.1 说明

本文件配套 CSV：

- `02_AI视频制作_模仿参考库标签字典_V0.1.csv`
- 标签总数：450
- 编码：UTF-8 with BOM，便于 Excel / WPS 直接打开
- 用途：作为 SQLite `tags` / `tag_aliases` / GUI 分面筛选 / Prompt Compiler / Reference Duty Planner 的初始受控词表

## 字段说明

| 字段 | 说明 |
|---|---|
| `tag_id` | 唯一标签 ID，自动生成 |
| `tag_key` | 稳定机器键，建议数据库使用 |
| `zh_name` / `en_name` | 中文 / 英文显示名 |
| `category` | L1 大类 |
| `parent_tag_key` | 上级标签 |
| `level` | L1 / L2 / L3 |
| `aliases_zh` / `aliases_en` | 中英文同义词、搜索别名 |
| `use_for_exact_filter` | 是否适合精确筛选 |
| `use_for_fulltext` | 是否适合全文搜索 |
| `use_for_semantic` | 是否适合语义搜索 |
| `use_for_weighting` | 是否适合进入反馈权重统计 |
| `default_usage_class` | 默认使用类别，如 offline_grammar_only / active_input_candidate / post_only |
| `default_active_input_allowed` | 默认是否允许 active input；除自有/授权候选外默认 false |
| `default_reference_role` | 默认参考职责，如 action_reference / camera_reference / fx_reference |
| `risk_level` | 风险级别 none / low / medium / high / critical |
| `description_zh` / `description_en` | 标签说明 |
| `examples` | 示例 |
| `do_not_use_for` | 不适用场景 |
| `source_basis` | 来源依据：TAG_SYSTEM_RESEARCH / SAFETY_RESEARCH / FX_POST_RESEARCH 等 |
| `status` | active_candidate / candidate 等 |

## 第一版使用建议

1. 全部视频导入时只需要粗标 L1/L2。
2. 高价值片段再补 L3。
3. `active_input_allowed` 默认仍应由片段级审核决定，不应只因为标签为 low risk 就自动放开。
4. 这个 CSV 是初始词表，不是最终不可变标准；后续应通过 GUI / SQLite migration 逐步增补。
