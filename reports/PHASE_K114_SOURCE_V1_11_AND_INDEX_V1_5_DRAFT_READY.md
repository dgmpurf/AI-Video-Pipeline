## 5. Summary Of V1.11 Rules

V1.11 draft adds:

- 5-second combat action-density minimum.
- First contact within `0.15s-0.30s`.
- No pure setup / leg movement longer than `0.35s`.
- At least `5-6` clear contact beats in a 5-second combat clip.
- Each contact beat must include attack path, contact point, defender reaction, rebound, and immediate continuation.
- Last `0.8s` must continue fighting and should cut mid-exchange.
- Actor-specific causality: use Fenshou / Shuangji explicit actions instead of generic "one character / another character."
- Environmental damage causality: exact impact must cause exact crack / splinter / debris.
- Local environment interaction should use one main spatial zone.
- Footwork must serve combat, not idle leg motion.
- Medium framing protects identity but does not guarantee action density.
- Speed-up is valid only when the action chain is already correct.
- Future combat prompt submit requires ChatGPT content audit after Codex package review and before human submit authorization.
