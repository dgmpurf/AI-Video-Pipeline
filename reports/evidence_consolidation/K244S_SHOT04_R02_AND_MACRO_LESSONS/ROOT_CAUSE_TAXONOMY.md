# SHOT-04 R02 Root Cause Taxonomy

## Purpose

This taxonomy names failure and route-decision classes observed or needed for SHOT-04 R02 evidence handling. It is evidence-only for ChatGPT Pro Extended Source synthesis. It is not official Source and does not authorize any Dreamina action, retry, resubmit, package edit, prompt edit, manifest edit, media creation, staging, commit, or push.

## Cross-Cutting Stop Rules

- If `sources/` is dirty, stop before evidence creation or live phases.
- If a submit returns `submit_id` but `gen_status=fail`, do not treat it as queued success.
- If two structurally distinct valid-submit routes fail remotely with no result media, stop retrying and route reset.
- If media exists, technical success is not visual success; build review artifacts and record human/ChatGPT review.
- No final/lock state can be inferred from package readiness, submit success, query success, download success, or review artifact creation.
